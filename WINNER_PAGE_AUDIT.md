Current page has a `Leveling plan`, but it is not explicit enough as a beginner progression framework.

Missing beginner progression:

- Level 1 decision: use one clear skill first.
- Early campaign decision: keep crossbow upgrades current.
- Rare monster decision: add single-target skill before damage feels bad.
- Safety decision: use Escape Shot before adding extra damage buttons.
- Complexity decision: add grenades only after the first loop is stable.

Recommended section:

`Beginner Progression`

Suggested steps:

1. `Start with clear`: use Fragmentation Rounds for packs.
2. `Add single-target`: use High Velocity Rounds for rares and bosses.
3. `Add safety`: keep Escape Shot ready before fights become crowded.
4. `Upgrade crossbow first`: weapon upgrades matter more than extra buttons early.
5. `Add grenades late`: use Gas Grenade or Explosive Grenade only after the core loop works.
